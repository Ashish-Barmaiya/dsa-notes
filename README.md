## Overview

This repository documents my journey of learning **Data Structures and Algorithms** in a structured way.

The objective is to develop a deep understanding of algorithms and data structures by studying:

* The underlying theory
* Internal implementation details
* Time and space complexity
* Trade-offs
* Problem-solving patterns
* Real-world applications
* Practical implementations in Go

Every topic is approached from first principles before moving on to coding problems.

---

## Philosophy

For every topic, I aim to answer the following questions:

* What problem does this solve?
* Why was it invented?
* How does it work internally?
* Why is its complexity what it is?
* What trade-offs does it make?
* How do I implement it from scratch?
* How is it used in production software?
* How do I recognize when to apply it?

The goal is to build intuition that transfers to unfamiliar problems rather than memorizing solutions.
